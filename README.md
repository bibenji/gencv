# generateur-de-cv

Générateur de CV conçu sur les technologies suivantes :
- HTML (avec utilisation de l'attribut "contenteditable"
- CSS (compilé avec SASS et COMPASS)
- JavaScript
- jQuery
- PHP

La librairie utilisée pour la génération de pdfs est fpdf :

http://www.fpdf.org/

Une version du projet est visible à l'adresse suivante :

www.generateurdecv.esy.es
